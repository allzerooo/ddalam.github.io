Lombok cannot find sybol 오류 해결 방법

September 06, 2019
Java Web Programming

SearchParam.java에 Lombok annotation을 사용했다. 코드는 에러 표시 없이 잘 활성화 되었고, 위 사진처럼 사용가능한 메서드에 Lombok annotation이 생성해준 메서드들이 잘 나왔다. 그런데 계속 cannot find symbol 오류가 발생했다🤨 더 멘붕인건 Lombok annotation이 생성해준 equals()는 문제없이 호출이 된다는 것이었다….

 

설정 과정

  1. IntelliJ - lombok 플러그인 설치
  2. build.gradle에 compile "org.projectlombok:lombok:1.18.8" 추가
  3. 설정 - Enable annotation processing 체크

 

Check 사항

  1. 설정 - Build, Execution, Deployment - Complier - Annotation Processors

Enable annotation processing에 체크했는지

  1. Lombok annotation을 사용했고, annotation으로 생성한 method들 중 getter, setter를 사용할 때 컴파일 오류가 발생하는지

 

해결 방법

Gradle build 파일 - dependencies에

annotationProcessor("org.projectlombok:lombok:1.18.8")

을 추가
