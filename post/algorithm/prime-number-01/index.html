어떤 수가 소수인지 아닌지 판별하는 방법

n이 소수가 아니라면,

루트 n 보다 작거나 같은 수 1개와 이에 대응하는 루트 n 보다 크거나 같은 수의 곱으로 표현할 수 있다. 따라서 루트 n까지만 검색해보면 어떤 수가 소수인지 아닌지 판별할 수 있다

def prime(n):
  if n < 2:
    return False
  i = 2
  while i*i <= n:
    if n % i == 0:
      return False
  return True
  • i는 정수인데 루트 n은 실수이기 때문에 정수-정수 비교를 위해 i*i ≤ n 식을 사용


범위에 있는 모든 수 중에서 소수를 찾는 방법

1부터 1,000,000까지 모든 수 중에서 소수를 찾는데 걸리는 시간 복잡도는, 각 수에 대해 O(루트n)의 시간으로 계산하면 너무 긴 시간을 필요로 한다.

                          1,000,000 * 1,000 = 10억 = 10초

따라서, 범위에서 모든 소수를 찾을 때는 에라토스테네스 체를 사용한다


에라토스테네스의 체

1부터 N까지 수 중에서 모든 소수를 구하는 방법

  1. 2부터 N까지 모든 수를 써놓는다
  2. 아직 지워지지 않은 수 중에서 가장 작은 수를 찾는다
  3. 그 수는 소수이다
  4. 그 수의 배수를 모두 지운다
  5. 수의 배수가 N을 넘지 않을 때까지 반복한다
  6. 마지막에 남아있는 수는 모두 소수이다


➕

선택된 수가 i라면, 그 수의 배수를 지울 때 i * i부터 지워나가면 된다.

왜냐하면 i * ( i 보다 작은 수 )는 i 보다 작은 수의 배수를 지울 때 ( i 보다 작은 수 ) * i 에서 지워지기 때문이다.

prime = []
check = [False] * (n+1)   # 지워졌으면 True

for i in range(2, n+1):
  if check[i] == False:
    prime.append(i)
    for j in range(j*j, n+1):
      check[j] = True
