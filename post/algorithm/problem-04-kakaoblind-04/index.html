from queue import PriorityQueue

def solution(food_times, k):
  answer = 0

  if sum(food_times) <= k:
    return -1

  # (food_times, index)를 사용해 우선순위 큐를 생성
  q = PriorityQueue()
  for i in range(len(food_times)):
    q.put((food_times[i], i+1))

  length = len(food_times)
  remaining_k = k
  prev = 0
  while (q.queue[0][0] - prev) * length <= remaining_k:
    remaining_k -= ((q.queue[0][0] - prev) * length)
    length -= 1
    prev = q.get()[0]

  # 큐에 남아있는 원소를 원래 순서대로 다시 정렬
  temp = sorted(q.queue, key=lambda x: x[1])
  target = remaining_k % length
  answer = temp[target][1]

  return answer


풀이

  • (food_times, index)의 값을 가지는 우선순위 큐를 생성한다
  • food_times를 정렬하면 최소 시간을 활용해 한 번에 많은 시간을 건너뛸 수 있다
    • 최소 시간 * length ≤ 남은 시간이면 최소 시간을 가지는 음식은 다 먹게된다
    • 나머지 음식들의 시간도 최소 시간만큼 줄어든다
    • 이때, 나머지 음식들의 시간을 줄일 때는 배열을 순회하면서 실제 값을 감소시키는게 아니라 정렬된 값을 활용한다
    • 1개의 음식을 다 먹었기 때문에 큐에서 제거해준다
  • 최소 시간 * length > 남은 시간이면 한 번에 건너뛸 수 없다. 따라서 while문에서 나오고, 남은 계산을 해야된다
  • 남은 시간 % length를 하면 다음으로 몇 번째에 있는 음식을 먹어야 되는지 계산할 수 있다
    • 이때 정렬된 배열이 아니라 원래 배열의 위치에 있는 음식을 먹어야 된다
    • 따라서 우선순위 큐를 인덱스를 기준으로 정렬한다
