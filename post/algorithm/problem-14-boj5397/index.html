BOJ_5397 키로거

September 23, 2019
Algorithm

문제


Code

testCase = int(input())
for _ in range(testCase):
  temp = []
  password = []
  inputPassword = list(input())

  for i in range(len(inputPassword)):

      if inputPassword[i] == "<":
        if password:
          temp.append(password.pop())
      elif inputPassword[i] == ">":
        if temp:
          password.append(temp.pop())
      elif inputPassword[i] == "-":
        if password:
          password.pop()
      else:
        password.append(inputPassword[i])

  for i in range(len(temp)-1, -1, -1):
    password.append(temp[i])

  print(''.join(password))


풀이

  • password 리스트의 끝을 커서로, 커서의 오른쪽을 temp 리스트라고 생각
  • 따라서 < 로 이동할 때는 password의 마지막 원소를 temp로
  • > 로 이동할 때는 temp의 원소를 password로 옮긴다
  • - 일 때는 password에서 원소를 하나 삭제한다
  • 마지막에 temp에 남아있는 원소들을 다 password에 추가해준다


보완할 부분

마지막에 temp에 남아있는 원소들을 password에 추가할 때 for문이 아니라 extend를 사용할 수 있다

password.extend(reversed(temp))
