BOJ_14391 종이 조각

September 18, 2019
Algorithm

문제


Code

n, m = map(int, input().split())
paper = [list(map(int, input())) for _ in range(n)]

answer = -1
for case in range(1<<(n*m)):
  sum = 0

  for i in range(n):
    num = 0
    for j in range(m):
      idx = i * m + j
      if (case & (1<<idx)) == 0:
        num = num * 10 + paper[i][j]
      else:
        sum += num
        num = 0
    sum += num

  for i in range(m):
    num = 0
    for j in range(n):
      idx = j * m + i
      if (case & (1<<idx)) != 0:
        num = num * 10 + paper[j][i]
      else:
        sum += num
        num = 0
    sum += num

  answer = max(answer, sum)

print(answer)


풀이

비트가 0일 때는 가로, 비트가 1일 때는 세로에 포함된다고 했을 때, n*m 개의 비트로 표현할 수 있는 모든 경우의 수에 대해서 가로, 세로의 합을 구해본다.

idx는 1차원으로 표현된 비트 값들을 입력으로 주어지는 가로, 세로 크기에 맞춰 2차원으로 표현했을 때 어떤 인덱스를 가지게 될 지를 나타낸다. (1<<idx)로 해당 인덱스의 비트를 1로 만들고, case와 & 연산을 했을 때 0이면 case의 idx번째 비트는 0인 것이고(가로), 0이 아니면 case의 idx번째 비트는 1인 것이다(세로).

이런 방법으로 가로에 포함되는지 세로에 포함되는지를 판단하고, 실제 종이 조각에 표현된 숫자를 사용해서 수를 구한다.
