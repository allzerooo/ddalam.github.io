IDE 없이 개발하는 방법

November 15, 2019
Java Web Programming


java 파일 1개를 생성, 컴파일, 실행하는 방법

  1. Main.java 파일을 생성

  2. 코드 작성

    class Main {
  public static void main(String[] args) {
    System.out.println("Hello World");
  }
}

  3. 컴파일 - Main.class 파일이 생성된다

    javac Main.java

  4. 실행

    java Main
    결과

Hello World



package 안에 있는 java 파일을 컴파일, 실행하는 방법

javac kr/co/company/Main.java
java -cp . kr.co.company.Main

-cp는 class path의 약자로 클래스를 찾기위한 경로를 말한다. 위 명령은 class path를 현재 디렉토리로 하고, package 아래 실행하고자 하는 class 파일을 지정해준 것이다.

결과

Hello World



jar 파일 생성, 실행하는 방법

클래스가 많아지면 보통 jar 파일로 묶어서 관리를 한다. jar 파일은 압축 파일로 아카이빙(여러 파일을 하나로 묶어서 관리)이 목적이다.

jar 파일 생성

jar -c -f company.jar kr/*

-c : create

-f : file name

-m : manifest 파일을 추가할 때 사용하는 옵션

jar 파일을 생성하면 manifest 파일이 생성된다. manifest 파일은 패키지, 파일 관련 정보를 저장하는 파일이다. 예를 들어, 패키지 안에 있는 파일은 다음과 같이 기록된다.

Main-Class: kr.co.company.Main
jar -c -m manifest.txt -f company.jar kr/*


jar 파일 실행 방법 1

java -cp company.jar kr.co.company.Main
결과

Hello World

class path로부터 실행하고 싶은 class 파일을 찾아서 실행하는 구조


jar 파일 실행 방법 2

jar를 만들 때 manifest 파일이 추가되어 있어야 가능한 방법이다

java -jar company.jar
결과

Hello World
