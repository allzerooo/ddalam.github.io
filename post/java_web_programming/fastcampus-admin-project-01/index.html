Spring Boot - admin 프로젝트 주제 & 목표

August 29, 2019
Java Web Programming

패스트캠퍼스 강의 내용을 정리


주제

Admin Site

  • 기존 Admin page의 부족한 기능 개발을 통해 서비스의 이해, 용어, 환경, 데이터베이스 구성, 비즈니스의 흐름을 파악 가능
  • 이러한 이해를 바탕으로 다른 프로젝트에 투입


목표

  1. 7개의 테이블이 있는 Admin Site를 개발
  2. 사용자가 쇼핑몰에서 아이템을 구매하거나, 관리자가 외부 업체를 관리하는 Admin Site
  3. 1000명의 유저와 구매 내역이 5000개가 넘는 데이터를 다룸
  4. Spring Boot를 사용
  5. 신규 주문, 신규 유저, 신규 파트너, 신규 상품을 볼 수 있는 대시보드를 제공
  6. 고객 관리, 상품 관리, 파트너사 관리, 주문 관리, 카테고리 관리, 어드민 사용자 관리 메뉴를 제공
